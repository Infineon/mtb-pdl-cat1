Voltage trip automotive compatible mode
GPIO (General Purpose Input Output) » Macros

General Description

Constants to be used for setting the input buffer automotive compatible mode (trip points and hysteresis) for GPIO upper bit.

Note
These macros are available for CAT1B devices.

Macros

#define CY_GPIO_VTRIP_DIS_AUTO   (0x00UL)
 Input buffer is not compatible with automotive and it is compatible with CMOS/TTL interfaces.
 
#define CY_GPIO_VTRIP_SEL_AUTO   (0x01UL)
 Input buffer is compatible with automotive.
