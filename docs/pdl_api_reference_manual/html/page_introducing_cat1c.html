Introducing CAT1C

CAT1C is an architecture platform which is designed to support the XMC7xxx family.

XMC7xxx is a high-performance, high quality grade MCU platform, targeting Industrial Drives, Robotics, Transportation(eBikes and LEVs), eV Chargers, Heavy duty powers and construction tools applications. It is the extension of the existing XMC4000 portfolio in terms of real-time performance, memory footprint and security features related to Industrial IoT.

XMC7xxx family is a family of MCUs with 32-bit ARM® Cortex®-M7 CPU for primary processing and an Arm® Cortex®-M0+ CPU for peripheral and security. There are variants with one CM7 core and variants with dual CM7 cores. The application sub-system is highly integrated with configurable serial communication blocks that can be turned into UART/I2C/SPI as needed, multiple timer/counter pulse-width modulators, I2S, PDM, CAN and LIN interfaces. XMC7xxx provides Hardware Secure Module (HSM) by CM0+ and Crypto engine. The HSM supports cryptographic functions and secure key storage. This function will be used for secure debugging, secure boot, secure Flashing and authenticated communication.

ARM® Cortex®-M7 CPU has internal instruction cache (I-Cache) and data cache (D-Cache). Cache coherency is supported with the ARM® CMSIS Level 1 Cache APIs for Armv7-M.

ModusToolbox™ will fully support XMC7xxx with low-level drivers, peripheral usage and application examples.
