Enables OVD over either VDDD or VDDA or VCCD.

Parameters
ovdSelcy_en_syspm_ovd_sel_t.
Note
This API is available for CAT1C devices.
OVD over VDDD and VCCD are enabled by default and cannot be disabled.

◆ Cy_SysPm_OvdDisable()

void Cy_SysPm_OvdDisable ( cy_en_syspm_ovd_sel_t  ovdSel)

Disables OVD over either VDDD or VDDA or VCCD.

Parameters
ovdSelcy_en_syspm_ovd_sel_t.
Note
This API is available for CAT1C devices.
OVD over VDDD and VCCD are enabled by default and cannot be disabled.
Disabling OVD over VDDA will perform the /ref cy_en_syspm_ovd_action_select_t action selected for VDDA. If no action is desired when disabling, firmware must first write CY_SYSPM_OVD_ACTION_NOTHING in a separate write cycle.
