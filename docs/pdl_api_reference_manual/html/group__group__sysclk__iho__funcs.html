◆ Cy_SysClk_IhoEnable()

void Cy_SysClk_IhoEnable ( void  )

Enables the IHO.

Note
Call SystemCoreClockUpdate after this function calling if it affects the CLK_HF0 frequency.
This API is available for devices having MXS40SSRSS and CY_IP_MXS22SRSS IP.
