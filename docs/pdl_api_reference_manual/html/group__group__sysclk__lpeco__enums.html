Note

Enumerator

Backup domain LPECO maximum amplitude is 1.35V.

Backup domain LPECO maximum amplitude is 1.8V.

Backup domain LPECO minimum amplitude detection.

Note
This enum is available for TVIIC devices.
Enumerator
CY_SYSCLK_BAK_LPECO_AMPDET_INIT_EN 

Initially enabled, and then automatically disabled when amplitude detector detects sufficient amplitude.

CY_SYSCLK_BAK_LPECO_AMPDET_ALWAYS_EN 

Keep minimum amplitude detector enabled as long as LPECO is enabled.
